Installation

Install the as-modal-dialogs bower component into your project directory 
bower install as-modal-dialogs

Use it in your web application

Inject the asModalDialogs module into your AngularJS application. 
angular.module('YourApp', ['asModalDialogs', /*Other dependencies*/]);
Import the .js and (optionally) .css files. Now you're ready to use the modal dialogs directives! 
<!DOCTYPE html>
<html ng-app="YourApp">
	<head>
		<script src="bower_components/angular/angular.min.js"></script>
		<script src="bower_components/as-modal-dialogs/as-modal-dialogs.min.js"></script>
		<link rel="stylesheet" href="bower_components/as-modal-dialogs/as-modal-dialogs.min.css" />
		<script src="js/your-app-file.js"></script>
	</head>
	<body>
		// Your HTML code

		<as-modal-container trigger="true">
			<as-modal-trigger display="block">
				Trigger text
			</as-modal-trigger>
			<as-modal-dialog modal-title="Modal Title">
				Modal dialog text.
			</as-modal-dialog>
		</as-modal-container>
	</body>
</html>
